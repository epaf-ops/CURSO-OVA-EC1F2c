Introducción a eXeLearning

ANTECEDENTES E INSTALACIÓN

eXeLearning permite integrar una variedad de recursos como Texto, Imágenes estáticas, Animaciones, Vídeo, Audio y posibilita la producción de actividades interactivas.  Los tipos de recursos interactivos disponibles son tipo Actividad desplegable, Lista desordenada, Pregunta de Elección Múltiple, Pregunta de Selección Múltiple, Pregunta de Verdadero-Falso, Rellenar huecos y Vídeo interactivo, entre otras opciones. Una de las principales ventajas de este software, es que se invierte poco tiempo para aprender a usarlo, es muy intuitivo.  

Una vez terminada la edición, es posible exportar el Objeto de Aprendizaje a formatos compatibles con una Plataforma Virtual de Aprendizaje para ponerlos a disposición de los estudiantes, opciones como Sitio Web (HTML), SCORM, ePub3, IMS.

Para instalar el software, entren a la página Web de  (https://exelearning.net/) y seleccionen el botón DESCARGAS para tener acceso a las versiones de software disponibles. Descarguen e instalen de acuerdo al sistema operativo donde van a utilizar el programa.

Al instalarlo en la computadora (Versión instalable), tendrán acceso directo a él a través del icono  que se genera en el escritorio como cualquier otra aplicación. Abran el programa y establezcan sus preferencias de Idioma y Navegador. Deben GUARDAR para que se mantengan esas preferencias. Pueden desactivar la opción para evitar que se visualice esa ventana de preferencias cada vez que entren al programa.
