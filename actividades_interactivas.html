Actividades interactivas

eXeLearning tiene varias opciones de actividades interactivas, de las cuáles, solamente el cuestionario SCORM es el único que puede almacenar el resultado final en el LMS, siempre y cuando sea insertado el objeto de aprendizaje en formato SCORM.  A pesar de que no se guarden las notas finales, este tipo de actividades desarrolladas en este software,  favorecen el aprendizaje de los estudiantes.

Entre las opciones se localizan la actividad desplegable, lista desordenada, pregunta de elección múltiple, pregunta verdadero-falso, rellenar huecos, vídeo interactivo, etc. En esta ocasión, daremos énfasis solamente a dos herramientas.

Vídeo Interactivo

Por medio de este IDevice se introduce una serie de preguntas a lo largo de un vídeo, pausando la reproducción hasta que el usuario lo determine. El siguiente material les permitirá conocer cómo retroalimentar el vídeo, crear un fotograma de forma paralela a él para mostrar información o generar una actividad para el usuario.

Actividad Geogebra

Este iDevice permite enlazar desde el objeto de aprendizaje a una actividad desarrollada en el software matemático Geogebra. Tiene apartados para establecer instrucciones y configuración de visualización. El siguiente material ofrece detalles para su implementación en eXeLearning.

NOTA

En este OVA se han abordado los elementos básicos de eXeLearning para que puedan generar su contenido de aprendizaje. Por lo cual, están preparados para iniciar a experimentar en él. Es muy intuitivo, por lo que los invitamos a explorar las distintas opciones que les ofrece.

Si tienen dudas respecto al uso de algunos de los elementos de eXeLearning, pueden consultar a la facilitadora del curso o ver la ayuda disponible en la página Web del software, es un TUTORIAL- MANUAL  completo que proporciona de forma detallada cada elemento que lo integra.
